pub struct NotificationTracker { /* HashSet-based dedup */ }
impl NotificationTracker {
    pub fn new() -> Self;
    pub fn is_notified(&self, key: &str) -> bool;
    pub fn mark_notified(&mut self, key: &str);
    pub fn reset(&mut self);
}

pub enum NotificationAction {
    AtTime { prayer: String, time_str: String },
    PreAlert { prayer: String, minutes: u32 },
}

pub fn check_notifications(
    prayer_list: &[(&str, DateTime<Local>)],
    config: &NotificationConfig,
    tracker: &NotificationTracker,
    now: DateTime<Local>,
    time_format: &str,
) -> Vec<NotificationAction>;

pub fn execute_action(
    action: &NotificationAction,
    config: &NotificationConfig,
    tracker: &mut NotificationTracker,
);

pub fn check_notify_send_available() -> bool;

**src/app.rs changes:**

1. Add fields to App struct:
   - `pub notification_tracker: notification::NotificationTracker`
   - `pub notification_config: config::NotificationConfig`
   - `pub prayer_time_message: Option<(String, chrono::DateTime<Local>)>` -- holds ("Maghrib", arrival_time) for "Prayer time!" display, cleared after 1 minute

2. Update `App::new()` to accept `NotificationConfig` parameter and initialize:
   - `notification_tracker: NotificationTracker::new()`
   - `notification_config` from parameter
   - `prayer_time_message: None`

3. Update `tick()`:
   - After midnight rollover recalculation, call `self.notification_tracker.reset()` to clear dedup state for new day
   - Call notification check: `let actions = notification::check_notifications(&self.prayer_list(), &self.notification_config, &self.notification_tracker, Local::now(), &self.time_format);`
   - Execute each action: `for action in actions { notification::execute_action(&action, &self.notification_config, &mut self.notification_tracker); }`
   - For AtTime actions, set `self.prayer_time_message = Some((prayer_name, Local::now()))` to trigger the "Prayer time!" display

4. Update `format_countdown()`:
   - Before the normal countdown, check `self.prayer_time_message`:
     - If Some((name, started)) and `Local::now() - started < Duration::seconds(60)`:
       Return `format!("{} \u{2014} Prayer time!", name)` (em dash per user decision)
     - If Some but elapsed >= 60s: clear to None, fall through to normal countdown
     - If None: normal countdown as before

5. Update `make_test_app()` in tests to include new fields (NotificationTracker::new(), default NotificationConfig)

**src/main.rs changes:**

1. After config merge and before App::new(), check notify-send availability:
   ```rust
   let mut notification_config = merged.notifications;
   if notification_config.desktop && !notification::check_notify_send_available() {
       eprintln!("Warning: notify-send not found. Desktop notifications disabled.");
       notification_config.desktop = false;
   }
   ```

2. Pass `notification_config` to `App::new()`:
   ```rust
   let app = app::App::new(prayers, merged.display.time_format.clone(), lat, lon,
       merged.calculation.method.clone(), merged.calculation.madhab.clone(),
       notification_config);
   ```
