    - Test: Sunrise is skipped for at-time notifications by default
    - Test: Pre-alert fires when current time is within pre_alert_minutes of prayer and not yet alerted
    - Test: At-time fires when current time >= prayer time and not yet notified

  <action>
Create src/notification.rs with:

1. **NotificationTracker** struct:
   - `notified: HashSet<String>` -- tracks "fajr_at", "fajr_pre" style keys
   - `fn is_notified(&self, key: &str) -> bool`
   - `fn mark_notified(&mut self, key: &str)`
   - `fn reset(&mut self)` -- clears all (called on midnight rollover)

2. **NotificationAction** enum:
   ```rust
   pub enum NotificationAction {
       AtTime { prayer: String, time_str: String },
       PreAlert { prayer: String, minutes: u32 },
   }
   ```

3. **check_notifications** function:
   - Takes `prayer_list: &[(&str, DateTime<Local>)]`, `config: &NotificationConfig`, `tracker: &NotificationTracker`, `now: DateTime<Local>`, `time_format: &str`
   - Returns `Vec<NotificationAction>`
   - For each prayer in list:
     - Skip Sunrise for at-time unless explicitly overridden (Sunrise pre_alert is already 0 by default)
     - Check at-time: `now >= prayer_time && now < prayer_time + 60s` and not already notified -> AtTime action
     - Check pre-alert: `prayer_time - pre_alert_minutes <= now < prayer_time` and not already pre-alerted and pre_alert_minutes > 0 -> PreAlert action

4. **execute_action** function:
   - Takes `action: &NotificationAction`, `config: &NotificationConfig`, `tracker: &mut NotificationTracker`
   - AtTime: send desktop notification (if config.desktop) + terminal bell (if config.bell), mark notified
   - PreAlert: send desktop notification only (if config.desktop), mark pre-alerted

5. **send_desktop** function:
   - At-time: `notify-send "Maghrib" "Prayer time: 6:23 PM"` (with mosque emoji in body per user decision: body is `"\u{1f54c} Prayer time: 6:23 PM"`)
   - Pre-alert: `notify-send "Maghrib" "Maghrib in 15 minutes"`
   - Uses `std::process::Command::new("notify-send")` with `.arg(title).arg(body)`

6. **send_bell** function:
   - Prints `\x07` to stdout (BEL character)

7. **check_notify_send_available** function:
   - Uses `std::process::Command::new("which").arg("notify-send")` to check availability
   - Returns bool

8. **Formatting helpers:**
   - `format_notification_body(time: DateTime<Local>, time_format: &str) -> String` -- returns "Prayer time: HH:MM" or "Prayer time: hh:mm AM/PM" based on format. Prefix with mosque emoji: "\u{1f54c} Prayer time: ..."
   - `format_pre_alert_body(prayer: &str, minutes: u32) -> String` -- returns "Maghrib in 15 minutes"

Add `mod notification;` to main.rs (module declaration only, no other changes to main.rs).
